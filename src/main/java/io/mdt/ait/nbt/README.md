# The AIT NBT Bible
## Position serialization:
`<root nbt>` -> `Pos`: `long`

## Dimension serialization:
`<root nbt>` -> `dimension` -> `name`: `String`, `location`: `String`

## Absolute Position serialization:
`<root nbt>` -> `Pos` -> `at`: `long`, `dimension` -> `name`: `String`, `location`: `String`
